METHOD 1: Drag & Drop (EASIEST)
--------------------------------
1. Start LDPlayer emulator
2. Drag the file "app\build\outputs\apk\debug\app-debug.apk" 
   into the LDPlayer window
3. Click "Install"
4. Open "My Application" in LDPlayer

METHOD 2: Using ADB (AUTOMATED)
--------------------------------
1. Start LDPlayer emulator
2. In LDPlayer:
   - Click menu (≡) in top right
   - Go to: Settings → Other settings
   - Enable "ADB debugging" (toggle ON)
   - Restart LDPlayer
3. Double-click "install-to-ldplayer.bat"
4. App will install automatically
5. Open "My Application" in LDPlayer

========================================
TROUBLESHOOTING:
========================================

If LDPlayer won't connect:
- Make sure ADB debugging is enabled
- Try port 5555, 5556, or 5557
- Restart LDPlayer
- Run: C:\Android\Sdk\platform-tools\adb.exe kill-server
  Then: C:\Android\Sdk\platform-tools\adb.exe start-server

If app crashes:
- Check LDPlayer Android version (needs Android 7.0+)
- Lower minSdk in app/build.gradle if needed
